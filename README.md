# blog-images

Hosted PNG diagrams for [architecture-weekly](https://github.com/Anilinfo2015/personal-notes/tree/main/architecture/weekly-digest) and [ai-weekly](https://github.com/Anilinfo2015/personal-notes/tree/main/AI/ai-weekly) digests.

Diagrams are rendered from Mermaid source via `mmdc` and pushed here so that Substack, Medium, Dev.to, and LinkedIn (which don't render Mermaid inline) can embed them as images.

URL pattern: `https://raw.githubusercontent.com/Anilinfo2015/blog-images/main/digest/<week>/<file-slug>-<N>.png`

Auto-managed by `.claude/skills/architecture-weekly/scripts/render-diagrams.py` in the parent repo.
